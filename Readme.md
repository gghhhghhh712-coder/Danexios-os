# 🌌 DANEXIOS OS 🌌
  
> **ARCHITETTURA WHATSAPP MD - MANUALE TECNICO DI DEPLOY E CONFIGURAZIONE**

</div>

---

### 🟢 1. ACQUISIZIONE SORGENTE (GITHUB)
* **Repository Forking:** Accedere al repository ufficiale [**Danexios-os**](https://github.com/gghhhghhh712-coder/Danexios-os). Eseguire il **Fork** tramite l'apposito comando nella barra degli strumenti superiore per generare un'istanza dedicata nel proprio profilo.
* **Asset Download:** All'interno del proprio fork, accedere al menu **Code** (tasto verde) e selezionare **Download ZIP**. Estrarre l'archivio localmente per la successiva fase di upload.

---

### 🔑 2. PROTOCOLLO DI AUTENTICAZIONE (SESSION ID)
* **Pairing Engine:** Accedere al portale [**Generatore Pair Code**](https://knight-bot-paircode.onrender.com/).
* **Data Injection:** Inserire il proprio numero di telefono completo di prefisso internazionale. Il sistema genererà e invierà il **Session ID** tramite **messaggio privato** direttamente sul tuo account WhatsApp.
* **Security:** Copiare la stringa ricevuta (es: `KnightBot!abc...`). Essa costituisce la chiave crittografica d'accesso al bot.

---

### 🚀 3. INFRASTRUTTURA KATABUMP (ORDINE E GESTIONE)
* **Pannello di Controllo:** Localizzare il menu laterale accanto all'icona del **razzo**. Le sezioni principali sono: *Dashboard, Order, Referral, Buy Credits*.
* **Configurazione Istanza:**
    * **Environment:** Selezionare **Node.js** (Parametro predefinito, **NON** modificare).
    * **Performance:** Sebbene esistano piani gratuiti, per un'operatività fluida sono raccomandati i piani Premium (1.50€ - 2.50€).
    * **Sistema a Gettoni:** Il servizio opera tramite gettoni mensili. Un investimento di 10€ garantisce circa **5 mesi** di uptime. È fondamentale monitorare il counter gettoni: all'esaurimento degli stessi, il server verrà sospeso.

---

### 🔐 4. CREDENZIALI DI ACCESSO (ID E PASSWORD)
**Procedura obbligatoria per lo sblocco del terminale di controllo:**
1.  **Recupero ID Utente:** Cliccare in alto a destra sull'icona del **Cerchio (Profilo)**. L'ID univoco è posizionato immediatamente sotto l'indirizzo email.
2.  **Identificazione Server:** Nella Dashboard, individuare la sezione **"Your Server"** e cliccare sull'istanza creata.
3.  **Security Key:** Accedere a **Action** -> **See**, scorrere a fine pagina e selezionare **Change Password**. Copiare la chiave alfanumerica visualizzata.
4.  **Terminal Login:** Cliccare su **Access Server (GO Server)** e inserire le credenziali (ID e Password) precedentemente ottenute.

---

### 📂 5. FILE SYSTEM E ALGORITMO `./../`
1.  Nel pannello di controllo, selezionare l'icona **Menu (≡)** e accedere alla sezione **Files**.
2.  **Upload:** Utilizzare la **freccia arancione** per caricare l'archivio ZIP di Danexios.
3.  **Extraction:** Cliccare sui tre puntini accanto allo ZIP e selezionare **Unarchive**.
4.  **Root Relocation (Spostamento):**
    * Entrare nella cartella estratta.
    * Marcare tutti i file cliccando sul **quadrato di selezione** accanto alla dicitura "Home".
    * Cliccare sull'icona della **cartella con la freccia** (posizionata vicino al cestino rosso).
    * Nel campo "File Name" digitare esattamente: `../`
    * Confermare con il tasto arancione **Move**. I file verranno proiettati correttamente nella directory principale.

---

### ⚙️ 6. CONFIGURAZIONE E START-UP
1.  Aprire il file **`config.js`**.
2.  **Owner Definition:** Inserire il proprio numero di telefono nel campo `ownerNumber`.
3.  **Session Mapping:** Incollare il Session ID nel campo `sessionID` (esattamente tra i delimitatori arancioni `| |`).
4.  **Data Save:** Cliccare sul tasto arancione **Save Content**.
5.  **Execution:** Tornare in **Console** e cliccare il tasto verde **Start**. Al log `clean session`, il sistema è operativo.

---

### 📜 7. LOGICA OPERATIVA E COMANDI
**Danexios OS** trasforma l'account WhatsApp in uno strumento di automazione avanzata. Le funzionalità core includono:
* **Group Management:** Moderazione automatica, sistemi anti-spam e gestione blacklist dinamica.
* **Multimedia Engine:** Download di contenuti in alta risoluzione da YouTube, TikTok (senza watermark), Instagram e Spotify.
* **Advanced Utilities:** Generazione istantanea di sticker, traduzioni multilingue e integrazione con motori di ricerca.
* **Comando Master:** Digitare `.menu` in qualsiasi chat per visualizzare l'interfaccia di controllo completa.
